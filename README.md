Description: We will be creating a study guide application that works in a similar fashion to the popular study guide website, Quizlet. In our application, users will be able to answer questions in different subjects/topics. They will also be graded on how well they answer the questions so that they have access to their results at the end.

The user is first greeted with a login page so that we can keep track of how many questions they answer correctly. Once they log in, they can choose which subject they would like to study. If the subject they desire to study in is not listed, there will also be an option to create your own quiz questions. However, if a user creates their own quiz questions, these would only be saved locally to their profile. Each of the subjects already provided will be sourcing its questions using an API.

Once the user finishes answering the quiz questions provided, they will see a results page. Within this page, they can see how many questions they answered correctly and incorrectly. Additionally, they can see which specific questions they answered incorrectly, so they can learn from their mistakes.
